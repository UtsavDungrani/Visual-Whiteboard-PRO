# Visual Whiteboard Pro

Minimal scaffold for Visual Whiteboard Pro.

Frontend (Vite + React) and Backend (Express) skeletons are created for local development.

Quick start

1. Install dependencies for frontend and backend:

```powershell
cd "d:\Projects\Visual Whiteboard Pro\frontend"
npm install
cd "d:\Projects\Visual Whiteboard Pro\server"
npm install
```

2. Run frontend (in one terminal) and backend (in another):

```powershell
cd "d:\Projects\Visual Whiteboard Pro\frontend"
npm run dev

cd "d:\Projects\Visual Whiteboard Pro\server"
npm start
```

Backend runs on `http://localhost:4000` by default. Frontend dev server runs on Vite default (usually `http://localhost:5173`).
